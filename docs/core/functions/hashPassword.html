    • hashPassword(password: string, salt?: string): PasswordHash

      使用scrypt算法对密码进行哈希处理（带盐值） 密码哈希是存储用户凭证的标准方式，避免明文存储密码

      Parameters

      • password: string

        待哈希的原始密码（明文）

      • salt: string = ...

        可选的盐值（若不提供则自动生成16字节的随机盐）

      Returns PasswordHash

      包含盐值和哈希结果的对象（盐值和哈希均为十六进制字符串） 注意：盐值需要与哈希结果一起存储（通常存入数据库），验证时需使用相同盐值

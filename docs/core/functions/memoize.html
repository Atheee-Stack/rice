    Function memoize

    • memoize<T extends unknown[], U>(fn: (...args: T) => U): (...args: T) => U

      创建记忆化版本的函数（Memoization） 记忆化通过缓存函数的历史调用结果，避免重复计算，提升性能 支持基本类型（自动序列化）和对象类型（通过WeakMap引用标识）的参数缓存

      Type Parameters

      • T extends unknown[]

        函数参数类型的元组（用于类型推断）

      • U

        函数返回值的类型

      Parameters

      • fn: (...args: T) => U

        要记忆化的原始函数

      Returns (...args: T) => U

      记忆化后的函数（相同参数调用时直接返回缓存结果）

      Example

      const factorial = memoize(n => n <= 1 ? 1 : n * factorial(n-1)); factorial(5) → 120（第二次调用更快）
